Modbus slave simulator in C

Dummy modbus slave server in C

Continuous Integration

I do continuous integration and build docker images after git push by self-hosted drone.io server for armhf platform , circleci server for x86 platform and dockerhub service.

CI Platform Status
Travis x86
CircleCI x86
Drone armhf

Pre-built docker images

You can download pre-built docker images from the dockerhub.

From source code

gcc server.c -o server -Wall -std=c99 `pkg-config --libs --cflags libmodbus`

From docker file (x86)

docker build -t c-modbus-slave:x86 .

From docker file (armhf)

docker build -t c-modbus-slave:armhf -f Dockerfile.armhf .
